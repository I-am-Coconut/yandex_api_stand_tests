# API-тесты для сервиса управления наборами (kits)
Набор автотестов для проверки API сервиса, отвечающего за создание пользовательских наборов (kits). Тесты покрывают валидацию поля name по различным сценариям.

# Структура проекта
create_kit_name_test.py — основные тестовые сценарии.
configuration.py — базовые URL и пути API.
sender_stand_request.py — функции отправки HTTP‑запросов.
data.py — тестовые данные и заголовки.

# Запуск тестов:
Установите зависимости: pip install pytest requests
Запустите тесты:
pytest create_kit_name_test.py -v

# Покрытые сценарии
Допустимая длина имени (1 и 511 символов).
Недопустимая длина (0 и 512 символов).
Поддержка различных символов:
английские и русские буквы;
спецсимволы;
пробелы;
цифры.
Проверка обработки отсутствующего параметра и неверного типа (число вместо строки).

# Предварительные требования
Доступ к сервису по URL из configuration.py.
Стабильное интернет‑соединение.

- Для запуска тестов должны быть установлены пакеты pytest и requests
- Запуск всех тестов выполняется командой pytest
